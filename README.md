# weapp-ui
针对微信小程序整合的一套UI库

## 目前样式 ##
### 列表
- 列表&列表项
- 带头像的列表项
- 带缩略图的列表项
- 列表间隔空间

### 表单

- 按钮三种大小
- 按钮六种颜色
- input默认样式
- input横向
- input堆叠

###布局

- row&col网格布局

说明文档请参考[wiki](https://github.com/liujians/weapp-ui/wiki "wiki")
